# Voice-Controlled Ollama Chatbot - Project Setup Guide

## Project Overview
A Python desktop application featuring:
- Ollama LLM integration for intelligent responses
- Voice command input with speech recognition
- Text-to-speech audio responses
- GUI interface with message history

## Technology Stack
- Python 3.8+
- Tkinter for GUI
- SpeechRecognition for voice input
- pyttsx3 for text-to-speech
- Ollama REST API for LLM
- Requests library for HTTP communication
